# Changelog

All notable changes to Toolkitty will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and future releases are expected to follow [Semantic Versioning](https://semver.org/spec/v2.0.0.html). Toolkitty does not yet have a supported stable release.

## [Unreleased]

### Added

- Contributor, support, security, conduct, ownership, issue, and pull request guidance for the maintained fork.
- A maintenance roadmap describing the recovery baseline and deliberately deferred work.
- Continuous integration, dependency update, and release automation for the maintained fork.

### Changed

- Project status and security expectations now describe the current pre-release alpha state.
- JavaScript dependencies and the Node, npm, and Rust toolchains were refreshed with reproducible lockfile-based workflows.
- Frontend types, forms, and API boundaries were updated for the maintained dependency set.

### Security

- Documented the private vulnerability-reporting path and current security limitations without claiming audit or release readiness.
